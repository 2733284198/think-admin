## 安装
根据情况选取适合自己版本，安装后端项目
```bash
# 最新源码
git clone https://github.com/edenleung/think-admin.git

# 稳定版
composer create-project xiaodi/tant

# 安装依赖
cd think-admin && composer install

# 执行安装
php think tant:install
```
